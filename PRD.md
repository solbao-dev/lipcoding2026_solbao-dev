# PRD.md - 개인 생산성 향상 앱


---

## ⚠️ 필수 규칙
1. 웹 앱 (HTML/CSS/JS)
2. **GitHub Copilot SDK (GitHub Models API) 를 앱 안에서 직접 사용**
3. Azure 클라우드 배포

## 앱 개요

- **앱 이름**: AI 생산성 도우미
- **한 줄 설명**: AI가 할 일을 분석하고 우선순위를 정해주는 스마트 생산성 앱
- **타겟 사용자**: 할 일이 많아서 뭐부터 해야 할지 모르는 직장인, 학생
- **핵심 가치**: 할 일을 입력하면 AI(Copilot SDK)가 우선순위와 예상 시간을 분석해서 오늘의 계획을 만들어줌
- **Copilot SDK 활용**: GitHub Models API로 AI 분석 기능 구현

---

## 핵심 기능 (MVP - 반드시 구현)

### 1. 할 일 입력
- 텍스트로 할 일 추가
- 여러 개 입력 가능
- localStorage 저장 (새로고침해도 유지)

### 2. 🤖 AI 분석 (Copilot SDK 핵심!)
- "AI 분석" 버튼 클릭 시 GitHub Models API 호출
- AI가 각 할 일의 **우선순위**, **예상 소요 시간**, **오늘 할 것 / 나중에 할 것** 분류
- 분석 결과를 카드 형태로 보여줌

### 3. GitHub Token 입력
- 앱 상단에 GitHub Personal Access Token 입력창
- 입력하면 API 호출 가능 (localStorage에 저장)

### 4. 할 일 완료 체크
- 완료 체크/해제
- 완료된 항목은 취소선으로 표시

---

## 추가 기능 (시간 남으면 구현)

- 타이머 시간 커스터마이징 (25분 기본, 변경 가능)
- 할 일에 카테고리 태그 추가
- 완료된 할 일 히스토리
- 배경 음악 / 백색소음 버튼

---

## UI/UX

- **색상**: 다크 테마 (배경 #1a1a2e, 포인트 #e94560)
- **레이아웃**: 중앙 정렬, 모바일 반응형
- **타이머**: 크고 선명하게 화면 중앙에
- **할 일 목록**: 타이머 아래 또는 오른쪽에 배치

## 기술 스펙

- **파일**: index.html 단일 파일
- **언어**: HTML + CSS + JavaScript (바닐라)
- **외부 라이브러리**: 없음 (필요시 CDN으로 최소한만)
- **데이터 저장**: localStorage
- **배포**: Azure Static Web Apps

## 앱 아이디어 후보 (세부 주제에 따라 교체)

세부 주제가 발표되면 아래 후보 중 하나를 골라 위의 "앱 개요 + 핵심 기능" 섹션을 교체하세요.

### 후보 A: 습관 트래커
- 매일 반복할 습관 등록 (운동, 독서, 물 마시기 등)
- 오늘의 습관 체크
- 연속 달성 일수 (streak) 표시
- 주간/월간 달성률 그래프

### 후보 B: 일일 다이어리 + 감정 기록
- 오늘의 감정 선택 (이모지)
- 한 줄 메모 기록
- 달력 뷰로 지난 기록 보기
- 월별 감정 통계

### 후보 C: 목표 달성 트래커
- 큰 목표 설정 (예: "책 1권 읽기")
- 진행 상황 퍼센트 표시
- 마일스톤 체크포인트
- 완료 축하 애니메이션

### 후보 D: 빠른 메모 + 태그 앱
- 빠르게 메모 입력
- 태그로 분류 (#업무 #개인 #아이디어)
- 태그별 필터링
- 검색 기능
