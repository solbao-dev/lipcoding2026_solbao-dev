# 🎤 천하제일 입코딩 대회 - 음성 코딩 가이드

## 🚀 대회 시작 직후 순서

1. **이 폴더를 VS Code로 열기**
   - VS Code 실행 → File → Open Folder → `대회준비` 폴더 선택

2. **Copilot Chat 열기**
   - 단축키: `Ctrl + Alt + I`
   - 또는 왼쪽 사이드바에서 채팅 아이콘 클릭

3. **음성 입력 시작**
   - Copilot Chat 입력창에서 **마이크 아이콘 클릭** (또는 `Ctrl + I`)
   - 말하기 시작!

---

## 🎙️ 음성으로 쓸 수 있는 슬래시 명령어

Copilot Chat에서 말하거나 입력하면 돼요:

| 말하면 되는 것 | 하는 일 |
|---|---|
| **슬래시 새앱** | 앱을 처음부터 만들기 |
| **슬래시 기능추가** | 현재 앱에 새 기능 넣기 |
| **슬래시 고쳐줘** | 오류 수정 |
| **슬래시 배포** | Azure에 배포하기 |

> 💡 슬래시 명령어는 `/새앱`, `/기능추가`, `/고쳐줘`, `/배포` 로 입력해도 됩니다

---

## 💬 이렇게 말하면 잘 됩니다

### 앱 만들기
```
"할 일 목록 앱 만들어줘. 할 일 추가하고 완료 체크할 수 있게"
"포모도로 타이머 앱 만들어줘. 25분 타이머에 시작 정지 버튼"
"습관 트래커 만들어줘. 매일 할 일 체크하는 앱"
```

### 기능 추가
```
"삭제 버튼 추가해줘"
"로컬스토리지에 저장되게 해줘"
"디자인 더 예쁘게 해줘"
"다크 모드 추가해줘"
```

### 문제 해결
```
"버튼이 안 눌려"
"화면이 이상하게 보여"
"콘솔에 에러 났어"
```

### 배포
```
"Azure에 배포해줘"
"배포 명령어 알려줘"
```

## ⌨️ 꼭 알아야 할 단축키

| 단축키 | 기능 |
|---|---|
| `Ctrl + Alt + I` | Copilot Chat 열기 |
| `Win + H` | Windows 음성 입력 (어디서든 사용 가능) |
| `Ctrl + Shift + P` | VS Code 명령 팔레트 열기 |
| `Ctrl + `` ` ` | 터미널 열기 |
