  • GitHub 토큰 → 로컬 스토리지에만 저장 (외부 전송 ❌)

  • 분석 결과 → 로컬 스토리지에만 저장 (서버 저장 ❌)

빠른 시작 4단계

  1. 1. GitHub 토큰 저장

    GitHub Personal Access Token을 입력하고 저장 버튼을 누르세요.

  2. 2. 할 일 1개 이상 추가

    작게 쪼갠 할 일을 넣어주세요. 예: 보고서 초안 쓰기, 발표 슬라이드 3장 수정.

  3. 3. 완료 체크와 삭제 테스트

    체크박스와 삭제 버튼이 정상 동작하는지 먼저 확인하세요.

  4. 4. AI 분석 실행

    미완료 할 일만 GitHub Models API로 보내서 우선순위와 예상 시간을 분석합니다.

현재 검증 상태

    1) GitHub 토큰 설정

    토큰은 이 브라우저 localStorage에 저장됩니다.

    처음 쓰는 분을 위한 3줄 빠른 안내

    • 1 GitHub Settings → Developer settings → Personal access tokens → Tokens (classic) 이동
    • 2 Generate new token (classic)에서 짧은 만료 기간 설정 후 발급
    • 3 Models 관련 최소 권한만 선택 후 발급한 토큰을 입력칸에 붙여넣기

    토큰은 비밀번호처럼 다루세요. 음성으로 읽거나 화면 공유로 노출하지 마세요.

    2) 할 일 추가

    음성 기능은 할 일 입력에만 사용됩니다. 토큰이나 민감한 정보는 말하지 마세요.

    음성 입력 대기 중

    처음 테스트할 때는 샘플 할 일 넣기 버튼으로 바로 검증할 수 있습니다.

      3) AI 분석

      심사자 / 테스트 안내

      실제 AI 분석은 GitHub 토큰 입력 후 실행할 수 있습니다. 토큰이 없으면 데모 분석 보기 버튼으로 AI 결과 화면과 추천 흐름을 바로 확인할 수 있습니다.
